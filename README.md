Toji no Miko
==============
![yysk](http://cdn.animeloop.org/files/gif_360p/5a5a17b5535a894e4eaf6194.gif)

Requirements
------------
- Unity2017.3.1f

Install
-------
```
git clone --recursive https://github.com/denx-jp/SSP.git
```
or
```
git clone https://github.com/denx-jp/SSP.git
git submodule init
git submodule update
```
